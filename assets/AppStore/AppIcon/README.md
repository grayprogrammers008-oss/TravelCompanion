# Pathio App Icon

## Placeholder Icon Instructions

For testing purposes, you can:

1. **Use an online generator:**
   - Go to [AppIcon.co](https://appicon.co)
   - Upload any 1024x1024 image (can be simple logo/text)
   - Download the generated icons
   - Place the 1024x1024 PNG here as `AppIcon-1024x1024.png`

2. **Quick Figma/Canva Design:**
   - Create 1024x1024 canvas
   - Background: Blue (#2196F3)
   - Add white "P" letter in center
   - Export as PNG
   - Save here as `AppIcon-1024x1024.png`

3. **Use provided placeholder:**
   - The icon generator script is available but requires PIL
   - Install: `pip3 install Pillow`
   - Run: `python3 generate_icon.py`

## Requirements

- Size: 1024 x 1024 pixels
- Format: PNG
- No transparency
- No rounded corners (Apple adds those)

## For Production Release

Create a professional icon with:
- Simple, recognizable design
- Works at all sizes (from 20px to 1024px)
- Represents travel/journey theme
- Matches Pathio brand colors
