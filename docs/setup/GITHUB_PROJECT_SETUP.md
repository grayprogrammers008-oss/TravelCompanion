## 🔄 Status Meanings

### Todo
- Issue is defined and ready to work on
- Not yet started
- Assigned team member hasn't begun

### In Progress
- Team member is actively working
- Branch created
- Code being written
- Should have related PR

### Done
- All subtasks checked off
- PR merged to main
- Feature is working
- Issue can be closed
